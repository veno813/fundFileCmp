# 基金导出文件比对工具
## 主要功能

* 比对两套TA系统导出文件数据是否一致。
* 可以对差异数据进行友好提醒。
* 对于多个销售商文件夹，仅勾选外层文件夹即可。
* 效率要求（执行完成时间）
***

## 对比规则
- [ ] 文件缺失
- [ ] 一方文件有值，一方文件为空
- [ ] 文件版本不一致
- [ ] 文件中数据条数不一样
- [ ] 文件中内容不一致
- [ ] 可以完整提示出内容不一致坐标
- [x] 区分销售商生成比对日志
